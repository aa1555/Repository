# 在终端窗口执行的命令

## 查看已经安装的库 

pip list

## 查看可更新的库

pip list --outdated

## 安装pip_review

pip install pip-review

>pip_review是用于更新所有库的第三方工具

## 用pip_review更新所有库

pip-review --local --interactive

## 使用Python

直接输入Python然后按回车。

## 更新库

python -m pip install --upgrade pip

>python -m pip：运行pip模块
>
>install --upgrade：更新已安装的包
>
>pip：要安装的包
